2. Transferir pasta instantclient_21_5 para o diretório
3. Transferir arquivo credentials.py para o diretório

5. Rodar <conda activate analise_callcenter>
6. Rodar <pip install -r requirements.txt>
7. Abrir crontab
8. Adicionar linha:<

>
